Verbos HTTP:
Os verbos do HTTP são uma parte importante do protocolo de transferência hipermídia (HTTP). Eles são usados para especificar o que o cliente deseja fazer com o servidor. Existem oito verbos HTTP:

GET: Obtém um recurso do servidor.
POST: Envia dados ao servidor para criar um novo recurso.
PUT: Atualiza um recurso existente no servidor, atualizando TODAS as informações obrigatoriamente.
PATCH: Atualiza um recurso existente no servidor, atualizando APENAS o campo que foi alterado.
DELETE: Exclui um recurso do servidor.

HEAD: Obtém uma versão sem corpo do recurso.
CONNECT: Estabelece uma conexão entre dois pontos finais.
OPTIONS: Obtém as opções disponíveis para um recurso.
TRACE: Envia o pedido de volta ao cliente.
Cada verbo tem seu próprio significado e uso específico. É importante usar o verbo correto para a tarefa que você está tentando realizar. Por exemplo, você deve usar o verbo GET para obter um recurso do servidor, o verbo POST para enviar dados para o servidor, o verbo PUT para atualizar um recurso existente e o verbo DELETE para excluir um recurso.

Os verbos HTTP são uma parte essencial do protocolo HTTP e são usados para garantir que as solicitações e respostas sejam enviadas e recebidas corretamente entre clientes e servidores.
